🌤 Containerized Weather App

This application fetches weather data from the external API, stores it in MySQL, and serves requests through Nginx.

Quick Weather Queries

  • London
  • Paris
  • New York
  • Seattle
  • Montreal

API Endpoints

  • Health Check
  • Weather History

How to Use

To fetch weather for any city, manually edit the URL: 

/weather/CityName

Examples:

  • /weather/Madrid
  • /weather/Berlin
  • /weather/Marrakech
